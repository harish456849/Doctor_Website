Introduction to Websites

A website is a collection of interconnected web pages hosted under a single unique domain name on a web server. This project connects HTML5 pages with Java servlets and JDBC to support doctor and patient registration and login.

Types of Websites

Websites can be informational, interactive, educational, commercial, or service based. In this hospital project, the pages work together as a service website where patients and doctors can register, log in, and share useful information through connected forms.
